Sistema operacional: Linux Mint
Linguagem: Python 3
Versão da linguagem: 3.6.7


Execução do analizador léxico: 

- O lexico utiliza arquivos de texto como entrada e saída.
- O arquivo de entrada é o 'in.txt', qualquer texto colocado nele será utilizado no analizador léxico.
- Executar a aplicação pelo terminal no diretorio onde o arquivo 'lexico.py' esta localizado, por exemplo: 
    ~/trabalho-1/lexico
- Execute a aplicação como qualquer outra aplicação feita em python, por exemplo: 
    python3 lexico.py
    
- A saída sera feita por arquivos de texto no mesmo diretorio, sendo os arquivos de saída "records.txt" e "hashtable.txt"
- Erros encontrados serão printados no terminal.



Arquivos e diretorios:
- Lista de palavras reservadas: "reserved_words.txt", listando as palavras reservadas, uma por linha, em letras minusculas
- Entrada: Arquivo de texto chamado "in.txt" no diretorio do arquivo main.py
- Saidas: Arquivos "records.txt" e "hashtable.txt", com a lista de records e a hashtable dos identificadores, respectivamente
- Outros arquivos para o funcionamento do automato: diretorio "lib" na mesma pasta do arquivo principal
